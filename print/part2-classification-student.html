STEP 4. 평가의 함정 — 정밀도와 재현율 (Phase 9)

앱페이즈 9에서 코로나 진단(2%) / 암 검진(0.5%) 두 시나리오의 임계값을 움직이며 지표를 관찰합니다.
정밀도 vs 재현율
▸ 정밀도 = TP / (TP + FP)  — 양성이라 외친 것 중 진짜 양성 비율. "헛발질이 적은가?"
▸ 재현율 = TP / (TP + FN)  — 진짜 양성 중 잡아낸 비율. "놓침이 적은가?"
→ 둘은 보통 한쪽을 올리면 다른 쪽이 내려가요(트레이드오프).
▲ 임계값을 좌우로 움직이면 TP/FP/FN/TN 비율이 모두 바뀐다
