STEP 1. 입시 합격 예측 — 가중치 추적

이 절의 목표
대학이 비공개한 합격 기준을 합격자/불합격자 명단의 점수로부터 거꾸로 추적할 수 있는지 직접 손계산으로 확인한다.
분류 = 0/1 결정 — 1부에서는 연속 값을 맞히는 회귀였다. 2부는 합격(1)/불합격(0)을 가르는 분류.
▸ 점수 = w₁·국 + w₂·수 + w₃·영 + w₄·탐 → 점수 > 컷오프면 합격, 점수 ≤ 컷오프면 불합격(앱과 동일한 규칙).
▸ 가중치(w)·컷오프는 비공개. 우리는 결과(합·불)만 보고 가중치 비율을 거꾸로 추정한다.
활동지 = 손계산용 축소판 — 앱 페이즈 6은 학생 10명으로, 페이즈 7은 데이터를 40명까지 늘려가며 학습합니다. 활동지에서는 손으로 직접 채워넣을 수 있도록 6명 → 30명으로 줄였어요. 비율과 패턴이 잘 보이도록 일부러 깔끔한 숫자를 골랐습니다(앱은 더 풍부한 데이터로 같은 절차를 진행합니다).

STEP 4. 평가의 함정 — 정밀도와 재현율

이 절의 목표
코로나·암 검진처럼 "놓치면 큰일 / 오인하면 큰일"의 비용 비대칭이 있는 분야에서 정밀도·재현율을 직접 계산해 어느 쪽을 우선해야 하는지 판단한다.
한 줄 약속 — "양성 = 이번에 찾고 싶은 것"
"양성(positive)"은 좋은 일이 아니라 이번에 우리가 찾아내고 싶은 사건이에요. 코로나에서는 감염자, 암 검진에서는 암 환자, 스팸 필터에서는 광고 메시지가 양성. 누가 양성인지 정해지면 TP·FP·FN·TN의 의미가 자동으로 결정됩니다.
정밀도 vs 재현율
▸ 정밀도(Precision) = TP / (TP + FP)  — 양성이라 외친 것 중 진짜 양성 비율. "헛발질이 적은가?"
▸ 재현율(Recall) = TP / (TP + FN)  — 진짜 양성 중 잡아낸 비율. "놓침이 적은가?"
→ 둘은 보통 한쪽을 올리면 다른 쪽이 내려간다(트레이드오프).
🎭 마피아 게임 비유 — 두 친구의 성격 차이
같은 마피아 게임을 한다고 합시다. 양성 = 마피아, 음성 = 시민. 모델은 "이 사람이 마피아야!"라고 지목하는 플레이어예요. 두 친구의 성격이 정밀도·재현율과 그대로 대응합니다.
  • 🧐 정밀도가 높은 친구 — "신중파"  확실할 때만 손가락질해요.
      👍 함부로 시민을 처형하지 않아 신뢰가 쌓임. 한 번 지목하면 다른 친구들도 따라줌.
      👎 의심돼도 증거 부족이라며 가만히 있다가 마피아를 살려두기 쉬움.
  • 🗣️ 재현율이 높은 친구 — "적극파"  조금만 수상해도 외쳐요.
      👍 마피아를 거의 놓치지 않아요. 결정적 순간에 망설이지 않음.
      👎 시민을 자꾸 잘못 처형해 마을이 텅 비고, 자꾸 틀리면 진짜 마피아를 지목해도 아무도 안 믿어줌(양치기 소년).
🤔 어느 쪽이 더 중요할까? 정답은 없어요. 마피아가 1명만 남았을 땐 정밀도가, 절반일 땐 재현율이 더 절실해요. 의료·면접·메시지 필터에서도 같은 질문 — 어느 비용이 더 큰가? 그게 임계값을 결정해요.
🎚️ 임계값(threshold)이란?
모델은 각 사람에게 0~1 사이 양성 확률을 출력해요(1에 가까울수록 양성 확실). 그런데 확률만으로는 양/음을 결정 못 하니, 어디서 자를지를 정해야 합니다 — 이 자르는 위치가 임계값. 예: 임계값 0.5 → 0.5 이상이면 양성. 임계값을 낮추면 양성이라 부르는 사람이 늘어나 재현율↑·정밀도↓, 높이면 그 반대.
