2부 — 분류와 평가

앱 페이즈 6~9 · 학생용과 동일 구조 + 정답·해설·관찰 루브릭
핵심 메시지 "정확도 한 숫자에 속지 말 것." 합·불, 양성·음성처럼 비대칭 비용을 가진 문제에서는 정밀도·재현율을 함께 봐야 한다는 감각을 심는 것이 2부 전체의 목표.

STEP 1. 입시 합격 예측 — 가중치 추적

활동지 운영학생용은 정시 시나리오 6명·시험 4명을 활동지 안에 모두 적어두어 앱 없이 자력으로 풀 수 있게 만들었다. 학생들이 W1(균등)을 직접 손계산해 정확도 83%를 발견하고, W2·W3 결과(미리 제공)와 비교해 "수학 가중치가 큼"을 거꾸로 추적하는 흐름.
1-1합격자/불합격자 패턴 — 모범 답안
예시합격 3명의 수학 점수는 8·9·9, 불합격 3명의 수학 점수는 5·5·6. 수학이 합·불을 가장 잘 가르는 변수로 보임. (국·영·탐은 합격자보다 불합격자에서 더 높게 나오는 경우도 있어 단독 변수로는 약함.)
1-2W1 (균등) — 정답 (점수 = 네 과목 평균)
학생점수 = 평균예측 (>7 합격, ≤7 불합격)실제맞음/틀림
김민서7.75합격합격✓
이지우7.25합격합격✓
박서연7.00불합격(경계, 7.0 그대로는 ≤이므로 불합격)합격✗
최도윤7.00불합격(경계)불합격✓
정하준6.75불합격불합격✓
강시우6.50불합격불합격✓
정답맞음 = 5개 → 정확도 = 5/6 ≈ 83.3%
자주 틀림박서연·최도윤이 점수 7.0으로 동률인데 한쪽은 합격·한쪽은 불합격이라 균등 가중치로는 구별 불가. 앱과 동일한 규칙(점수 > 컷오프)이므로 7.0은 모두 불합격으로 예측 → 박서연 한 명을 틀린다.
1-3가중치 비교 + 비밀 공식 추적 — 정답
정답가장 정확한 가중치는 W2 (수학 강조)로 100%. W1(균등) 83% > W3(영어 강조) 33%.
비밀 공식 추적 — 모범 답안"여러 가중치 시도 결과의 정확도 차이를 비교하면, 가장 높은 정확도를 내는 비율이 진짜 합격 공식과 가장 가까울 것이다." 또는 "가중치 차이가 정확도 차이로 드러나기 때문에, 정확도 패턴을 보고 어느 변수가 큰지 추정한다."
수업 팁"비밀 공식을 정확히 알 수는 없어도, 결과로부터 비율 패턴을 좁힐 수 있다"는 점이 통계·머신러닝의 핵심 발상임을 짚을 것.
1-4학습 vs 시험 — 정답
정답시험 4명 중 맞음 = 3 → 시험 정확도 = 3/4 = 75%. (틀린 경우: 장이안 — W2로 6.5점이라 불합격 예측이지만 실제는 합격.)
차이 원인 — 모범 답안학습 6명에서 100%였던 가중치가 시험 4명에서 75%로 떨어진 이유: 학습 데이터가 너무 적어 그 6명의 우연한 패턴까지 외웠기 때문. → 과적합(overfitting)의 직관적 설명. 새 학생 장이안처럼 "수학은 약하지만 다른 과목 우수"인 합격자는 W2로는 못 잡음.
자주 틀림"학습 정확도가 높을수록 좋은 모델"이라는 오해. 실제 평가는 시험 정확도로만 한다는 점을 강조.

STEP 2. 데이터 추가 후 재학습

2-16명 → 30명 결과 — 정답
정답학습 정확도는 100% → 93%로 내려갔다. 시험 정확도는 75% → 90%로 올라갔다. 두 정확도의 격차는 25%p → 3%p로 줄었다.
의미학습 데이터가 많아지면 "우연한 패턴 외우기"가 줄고 진짜 규칙이 드러난다. 시험에서 더 잘 맞히는 일반화로 이어진다.
수업 팁학생이 "학습 정확도가 떨어진 게 좋은 일?"이라고 의아해하면 OK 신호. "학습 정확도 = 외운 정도, 시험 정확도 = 진짜 실력"임을 짚어줄 것.
2-2가상 인물 P2 vs P3 — 모범 답안
예시P2(수학만 약점)는 사람이 보면 합격에 가깝게 보이지만, 모델은 7.0 경계로 합격(W2 수학 강조라 P2가 손해 봄). P3(수학만 강점)은 사람이 보면 합격에 가깝지 않지만, 모델은 7.0 경계로 합격으로 판정 — 수학 가중치가 크기 때문. 모델이 "수학 가중치가 크다"는 학습된 패턴을 그대로 적용해 사람의 직관과 다른 판정을 낼 수 있음을 보여주는 사례.
자주 틀림P2·P3의 점수가 같으니 "둘 다 같다"고만 답하는 경우. 같은 점수라도 점수 분포(어디가 강점이냐)가 모델 해석에 어떻게 영향을 주는지 짚을 것.
2-3일반화 — 모범 답안
예시학습 정확도가 떨어진 건 "외운 패턴이 부서진" 신호이고, 시험 정확도가 오른 건 "진짜 규칙에 가까워진" 신호다. 일반화 = 처음 보는 데이터에서도 맞히는 능력이라는 정의에 맞으므로 좋은 신호.
전이 메시지 "데이터가 곧 권력이다." 이 직관이 거대 모델 시대의 핵심.

STEP 3. 정확도와 혼동행렬

3-1·3-2혼동행렬·정확도 — 정답
정답 실제 합계: 합격 8+2 = 10명, 불합격 1+9 = 10명. 총 20명.
TP = 8, FP = 1, FN = 2, TN = 9.
정확도 = (8 + 9) / 20 = 17/20 = 85%.
3-3정확도 함정 — 정답
정답전원 불합격이라 외치면 TP=0, TN=99, FP=0, FN=1. 정확도 = (0 + 99) / 100 = 99%.
비판 예시합격자 1명을 100% 놓치는데도 정확도는 99%. 정확도 한 숫자만으로는 "쓸모"를 평가할 수 없다.

STEP 4. 평가의 함정 — 정밀도·재현율

앱 운영Phase 9가 마피아 게임 비유(🧐 정밀도 = 신중파, 🗣️ 재현율 = 적극파)와 임계값 누적 히스토그램(임계값 오른쪽이 양성 판정 영역으로 음영)으로 재설계됐다. 활동지에는 마피아 비유 박스를 본문에 옮겨두었으니, 학생이 활동지의 비유로 정밀도·재현율의 성격을 먼저 파악한 뒤 앱에서 임계값 슬라이더를 직접 움직이며 두 친구의 변화를 관찰하게 운영하면 좋다. "양성 = 이번에 찾고 싶은 것"이라는 한 줄 약속을 칠판에 써둘 것 — 시나리오마다 양성의 정의가 바뀌어 헷갈리는 학생이 많다.
수업 팁마피아 게임 경험이 있는 학급이라면 도입을 "이 친구는 어느 쪽에 가까워?"로 시작. 한 친구가 무조건 옳은 게 아니라, 마을의 상황(마피아가 몇 명 남았는지)에 따라 답이 달라진다는 점이 임계값을 결정하는 비용 비대칭과 정확히 같음을 짚는다.
4-1코로나 — 정답 (소수점 둘째 자리)
임계값정확도정밀도재현율
0.387.9%19/139 ≈ 0.1419/20 = 0.95
0.596.6%16/46 ≈ 0.3516/20 = 0.80
0.798.5%10/15 ≈ 0.6710/20 = 0.50
트레이드오프임계값을 낮출수록 재현율↑(놓침 적음), 정밀도↓(헛발질 많음). 정확도는 0.5 부근에서 최대.
자주 틀림정확도가 99%에 가까운데 모델이 좋다고 단순화. 재현율(50~95%)을 함께 봐야 한다는 점을 짚을 것.
4-2코로나 임계값 결정 — 모범 답안
정답FN(놓침) 비용 ≫ FP 비용 → 재현율 우선 → 임계값을 낮춘다(예: 0.3). 약간의 오인 격리는 감수하고 환자를 놓치지 않는 쪽.
4-3암 검진 — 정답 (소수점 둘째 자리)
임계값정밀도재현율
0.35/85 ≈ 0.065/5 = 1.00
0.54/24 ≈ 0.174/5 = 0.80
0.72/5 = 0.402/5 = 0.40
정답코로나와 같은 방향: 놓침(FN)이 더 큰 비용 → 재현율 우선. 다만 암은 추가 검사 비용·환자 불안 부담이 있어 임계값을 무한정 낮추진 못함. 실제 의료의 패턴은 1차 검진 재현율 우선 + 2차 정밀검사로 정밀도 회복의 단계적 검진.
전이"단계적 검진 = 임계값을 다르게 둔 모델 두 개의 직렬 연결"이라는 메타 메시지를 함께 짚어주면 좋음.
4-4모두 음성 모델 — 정답
계산 재현율 = TP / (TP + FN) = 0 / (0 + 5) = 0%
정밀도 = TP / (TP + FP) = 0 / 0 → 정의 불가 (양성이라 외친 적이 없으므로 분모 = 0).
의미이 모델은 양성 환자를 단 한 명도 못 잡았고(재현율 0), "양성"이라는 판정 자체를 한 적이 없어 정밀도는 정의되지 않는다. 정확도 99.5%라는 숫자가 얼마나 거짓 위안인지 드러난다.
자주 틀림정밀도를 0%로 답하는 경우. 0/0은 0이 아니라 "정의 불가"임을 짚을 것.
4-5시나리오 미니퀴즈 — 정답
💼 회사 면접 — 선택별 효과 재현율 ↑ (천재 놓치지 않기) → 임계값을 낮춤 → 의심스러워도 일단 통과 → 천재는 거의 다 합격, 빌런도 일부 통과(FP↑).
정밀도 ↑ (빌런 거르기) → 임계값을 높임 → 확실한 사람만 통과 → 빌런은 거의 차단, 천재도 일부 떨어짐(FN↑).
현실 해법 — 모범 답안1차는 재현율↑(서류 넓게), 2차는 정밀도↑(면접 깐깐하게). 다단계 채용이 임계값을 다르게 둔 모델 두 개를 직렬 연결한 구조와 같다.
💬 메시지 스팸 필터 — 모범 답안대부분의 메시지 앱은 정밀도 우선. 친구의 중요한 안내(음성)를 광고함에 가두는 비용이 광고 한두 개 새는 비용보다 훨씬 크기 때문. (단체 톡방의 광고 차단보다는 일반 메일의 스팸 필터에서 더 엄격한 정책을 보임.)
전이4-5는 마피아 비유의 마지막 풀어내기. "마을의 상황(=문제 상황)에 따라 답이 다르다"는 한 줄을 학생이 자기 말로 옮길 수 있으면 4부 마무리. 다음 5부(생성)에서 평가 지표는 잠시 사라지지만, 6부(언어)에서 "다음 토큰 분포 평가"로 다시 돌아온다는 점 예고.
자주 틀림두 시나리오 모두 "재현율 우선"으로 답하는 경우. 메시지 필터의 음성(정상 메시지) 손실이 양성(광고) 누락보다 비싸다는 점을 정성적으로 짚어줄 것.

보충 활동 ★. 인간 혼동행렬 — 카드로 직접 나누기 (정답)

활동 의도공식 TP/(TP+FP)를 외우는 대신, "양성이라 부른 묶음"과 "실제 양성인 묶음"을 손으로 나눠 분자·분모의 출처를 직관으로 잡게 한다. 모둠당 카드 20장이면 5분, 임계값 변화 ★4까지 묶어 10~15분 활동.
★1카드 구성 — 권장 분포 (TP=6, FN=3, FP=4, TN=7)
정답학생용 활동지에 인쇄된 분포 그대로 운영. 권장 색지 — 합격은 흰색, 불합격은 노란색. 한 면(앞)은 실제, 뒷면은 예측. 카드 크기는 A6(105×148mm) 절반 정도.
운영 팁모둠당 1세트(20장) 미리 절단해 봉투에 넣어 두면 활동 시간이 절반으로 줄어듦. 카드 분포는 일부러 "정확도는 높은데 정밀도는 그저 그런" 비대칭으로 골랐다 — 마지막 ★3에서 정확도와 정밀도가 다른 숫자가 나오는 게 학습 포인트.
★2두 묶음 — 정답
정답 ⓐ "예측 = 합격"카드 수 = TP + FP = 6 + 4 = 10장. 그중 실제도 합격(TP) = 6장. 정밀도 = 6 / 10 = 60%.
정답 ⓑ "실제 = 합격"카드 수 = TP + FN = 6 + 3 = 9장. 그중 모델도 합격이라 부른 카드(TP) = 6장. 재현율 = 6 / 9 ≈ 66.7%.
학생 발견두 묶음의 분자가 똑같이 TP=6인데 분모가 다르다는 사실을 학생이 직접 발견하게 두기. "분모만 다르다"는 한 줄이 P9 9.4의 Aside와 정확히 일치.
★3혼동행렬·다섯 지표 — 정답
정답
실제 합격실제 불합격
예측 합격TP = 6FP = 4
예측 불합격FN = 3TN = 7
정확도 = (6 + 7) / 20 = 65% · 정밀도 = 6 / 10 = 60% · 재현율 = 6 / 9 ≈ 66.7% · F1 = 2·0.6·0.667 / (0.6 + 0.667) ≈ 63.2%.
관찰: 묶음 ⓐ ∩ ⓑ = TP(=6장). 정밀도와 재현율이 같은 분자(TP)를 공유한다는 사실은 두 묶음의 교집합이라는 시각적 표현으로 가장 잘 잡힌다.
자주 틀림① F1을 산술평균(0.633)으로 답 — 우연히 비슷한 숫자라 발견이 어려움. 정밀도가 낮은 케이스로 옮겨 차이를 보이거나, 단순 산술평균과 조화평균이 다른 식임을 짚어줄 것. ② 묶음 ⓐ ∩ ⓑ = "FP" 또는 "FN"으로 답 — 두 묶음에 모두 들어가려면 "예측도 합격 + 실제도 합격" → TP뿐이라는 점 강조.

임계값 ↓늘늘(또는 같음)늘↓↑
핵심: 임계값을 올리면 "양성이라 부른 카드"가 줄어드니 묶음 ⓐ가 줄고, FP가 먼저 빠져 정밀도가 오른다. 동시에 진짜 양성도 일부 놓치게 되어 TP가 줄고 재현율이 떨어진다.
전이이 표가 그대로 P9 9.6의 "상황이 바뀌면 우선순위가 바뀐다"의 정량적 근거가 된다. 또 Phase 9 시뮬레이터에서 슬라이더를 움직이면서 metric 카드의 분자·분모(예: 19/39)가 바로 이 표대로 변하는 모습을 함께 보여 줄 것.

STEP 5. 마무리 — 지표 선택 가이드

정리정답 매트릭스
상황가장 중요한 지표이유
스팸 메일 분류정밀도정상 메일을 스팸함으로 보내면(FP) 사용자가 중요한 메일을 놓침. FP 비용이 크다.
암 조기검진재현율진짜 환자를 놓치면(FN) 생존율 급락. FN 비용이 가장 크다.
입시 합격 예측정확도(또는 균형 F1)합·불 비율이 비슷하고 두 오류의 비용이 비슷하면 정확도가 무난.

관찰·수행 평가 루브릭

수준분류 모델 조작혼동행렬·지표지표 선택 판단
A 도달 한 번에 한 변수씩 조정해 시험 정확도를 80% 이상으로 끌어올림. 학습/시험 정확도 차이를 일반화 개념으로 설명. 정확도·정밀도·재현율을 모두 정의에 따라 정확히 계산. 임계값 변화의 트레이드오프를 자기 말로 설명. 3개 시나리오 모두에서 비용 구조에 맞는 지표를 정확히 골라 이유 설명.
B 진행 가중치 조정으로 70% 부근까지 도달. 두 정확도 차이의 의미는 모호. 정확도는 계산하지만 정밀도·재현율 분모를 헷갈림. 스팸·암 중 한 쪽은 맞히지만 이유가 단편적.
C 보충 슬라이더를 무작위로 움직여 정확도가 50%대. TP/FP/FN/TN의 의미를 구분하지 못함. "정확도 높으면 좋은 모델"이라는 인식에 머무름.
전이 포인트2부에서 다룬 모든 분류는 점수 계산 + 임계값 비교의 단일 뉴런이었다. 3부에서 같은 도구로 픽셀 단위 도트 그림을 분류하면서 "특징이 많아지면 어떻게 되는가"를 묻게 된다.
