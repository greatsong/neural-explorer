5부 — 분류를 넘어 생성으로

앱 페이즈 13~14 · 학생용과 동일 구조 + 정답·해설
핵심 메시지 5부의 한 문장: "분류기는 한 점이 어느 영역에 속하는지를 묻고, 생성기는 어느 영역의 한 점을 가져다 그림으로 펼친다." 학생이 평균/보간/잠재 공간이라는 세 도구의 역할을 자기 말로 정리할 수 있어야 한다.

STEP 1. 평균과 분포 — 가장 단순한 생성 모델

활동지 운영학생용은 두 라벨(가로줄 A·세로줄 B) 평균 패턴을 미리 그려 제공하고, 보간 G(blend) = (1−blend)·A + blend·B 식의 가중평균을 한 픽셀씩 직접 계산하게 만들었다. 정수 픽셀(0/1) + blend 0/0.5/1로 단순화해 손계산 부담을 최소화.
1-1평균이 곧 중심 — 모범 답안
예시"여러 장 그림에서 각 픽셀이 켜진 횟수를 세어 평균낸 것이 평균 이미지. 그래서 가장 일반적이고 흔한 모양 = 그 라벨의 중심." 또는 "분포의 중심점."
1-2분산의 의미 — 모범 답안
예시분산이 큰 자리 = 학생마다 칠했다 안 칠했다 변화가 큰 자리. A 라벨에서 가로줄을 0·1행에 그린 학생도 있고 1·2행에 그린 학생도 있다면, 1·2·3행 경계 근처가 가장 분산이 크다. 즉 같은 라벨 안에서도 다양성이 생기는 영역.
수업 팁"평균이 곧 생성"이라는 말은 다소 도발적. "분포의 중심점을 새 샘플로 가져다 쓴다"가 정확한 표현이라는 보충.
1-3보간 — 정답
0행 0열 (A=1, B=1, 둘 다 켜짐) blend = 0 → G = 1·1 + 0·1 = 1
blend = 0.5 → G = 0.5·1 + 0.5·1 = 1
blend = 1 → G = 0·1 + 1·1 = 1
0행 4열 (A=1, B=0, A에서만 켜짐) blend = 0 → G = 1·1 + 0·0 = 1
blend = 0.5 → G = 0.5·1 + 0.5·0 = 0.5 (중간값)
blend = 1 → G = 0·1 + 1·0 = 0
중간이 어중간한 이유 — 모범 답안blend = 0.5 그림은 두 라벨의 평균을 정확히 반반 섞은 것. A에만 있던 픽셀과 B에만 있던 픽셀이 모두 0.5 같은 중간값이 되어, 어느 라벨로도 분명하게 분류할 수 없는 어중간한 모양이 된다.
자주 틀림"중간은 그냥 두 그림을 겹친 거다"로만 답하는 경우. "중간값(0.5)이 만들어내는 흐릿함"이 핵심임을 짚기.
1-4노이즈 = 다양성 — 모범 답안
정답"평균(중심)에서 분산(허용 오차)만큼 벗어난 다른 점들을 매번 다시 뽑아내 새 그림을 만드는 일." 즉 같은 분포 안에서 서로 다른 샘플을 생성하는 능력.
전이이 직관이 그대로 VAE의 sampling으로 이어진다 — 평균과 분산을 학습해 그 안에서 점을 뽑는다. 5부 마지막에서 짧게 언급.

STEP 2. 오토인코더 — 잠재 공간으로

2-1구조 — 정답
입력(64) → 인코더 → 잠재(2차원) → 디코더 → 출력(64픽셀)
학습 목표: 입력과 출력이 같아지도록 W·b를 찾기 (= 자기 자신 복원)
분류와의 차이분류는 정답 라벨이 외부에서 주어지지만, 오토인코더는 입력 자체가 정답(self-supervised). 라벨이 필요 없다.
수업 팁"라벨이 필요 없는 학습"은 거대 모델(LLM·SSL)의 핵심 발상이라는 점을 짧게 짚어둘 것 — 6부의 GPT 학습으로 이어지는 다리.
2-2잠재 공간 탐험 — 정답
정답 z₁ 축 의미 = 라벨 A에서 B로의 변환 다이얼(왼쪽 끝 → A, 오른쪽 끝 → B).
z₂ 축 의미 = (학습된 보조 특성) 진하기·흐릿함·세부 패턴 등. 9칸 표에서 가운데 행(z₁=0)일수록 라벨 구분이 흐릿해진다.
한 줄 정리"잠재 좌표 = 의미의 다이얼이며, 한 축이 한 가지 의미 차원을 잡고 있다." z₁이 커질수록 A → B로 의미가 점진적으로 변환된다.
참고실제 학습 결과는 시드에 따라 축의 의미가 달라질 수 있음. 활동지 표는 "z₁ = A↔B 다이얼"이라고 가정한 단순화된 결과를 미리 제공한 것.
2-3학습 외 좌표 — 모범 답안
정답"디코더는 학습 데이터가 모인 영역(z₁·z₂가 [−1.5, +1.5] 안)에서만 의미 있는 그림을 복원하도록 학습됐다. 그 영역 밖은 한 번도 본 적 없는 입력이라 출력 품질이 보장되지 않는다." 즉 학습된 적 없는 좌표 = 디코더의 외삽 영역.
전이이것이 "왜 좋은 잠재 공간을 만들기 어려운가"의 핵심. VAE·정규화 같은 후속 기법은 모두 잠재 공간을 매끈하게 채우려는 시도로 이해할 수 있음.

수준평균·분포 (Phase 13)오토인코더 (Phase 14)분류 vs 생성 정리
A 도달 평균/분산의 의미를 자기 말로 설명. 보간 중간점이 어중간해지는 이유를 결정 경계로 설명. 인코더–잠재–디코더 구조와 자기 복원 학습 목표를 정확히 설명. 잠재 공간 탐험에서 축의 의미를 발견. 4가지 항목 모두 정확. "분류기를 뒤집으면 생성기"라는 말의 의미를 잠재 공간으로 설명.
B 진행 평균과 분산은 구분하지만 분산의 의미가 모호. 구조는 그릴 수 있으나 "자기 복원"의 학습 목표를 분류와 헷갈림. 표는 채우지만 두 패러다임의 차이를 한 문장으로 압축하지 못함.
C 보충 평균이 왜 "생성"인지 못 설명. 잠재 공간이 무엇인지 인지하지 못함. 분류와 생성을 입력·출력 차원에서 구분하지 못함.
전이 포인트5부까지는 모두 그림이 입력이었다. 6부에서는 입력이 텍스트로 바뀌고, "글자도 결국 숫자다"라는 메시지를 ASCII·토큰·임베딩 순으로 풀어간다. 잠재 공간 ≈ 단어 임베딩 공간이라는 연결고리를 미리 던져둘 것.
