A — 알고리즘의 이해

예측 → 오차 → 기울기 → 갱신을 한 묶음으로 (학생용과 동일 구조 + 정답)
핵심 메시지 학생이 A 영역(A1~A5)을 마칠 때 "학습 = 손실을 줄이는 방향으로 w·b를 한 step 만큼 미는 일을 반복"을 자기 말로 말할 수 있어야 한다. 수식보다 한 사이클의 흐름에 집중시킬 것.
앱 운영A2~A5 가 같은 데이터(y = 2x + 1, 5점)를 공유하도록 통일했음. 학생이 A2 에서 슬라이더로 직접 찾던 (w, b)를 A3~A5 가 자동으로 찾아 가는 흐름임을 칠판 한 줄로 먼저 적어 둘 것: "손으로 한 일을 컴퓨터가 자동으로 한다". ½MSE 규약으로 통일 — 손실 L = ½ · MSE 라고 표기.

A2-5왜 제곱? — 정답
  1. 부호 상쇄 방지 — +3 + (−3) = 0 → "오차 없음"처럼 보임. 제곱하면 모두 양수가 됨.
  2. 큰 오차에 더 큰 페널티 — 1²=1, 10²=100. 큰 실수가 폭발적으로 커진다.
  3. 매끄러운 곡선 — e² 은 포물선(2차 곡선) 모양이라 한 점에서 기울기를 정확히 잴 수 있음.
서술 예시매끄러운 곡선이라야 "어느 방향으로 얼마나 옮길지"의 기울기 정보가 매 점에서 살아 있어, 다음 STEP 의 경사하강법이 작동한다. 절댓값 |e| 는 0 에서 꺾여 미분이 정의되지 않아 같은 일을 못 함.
수업 팁"꺾인 함수 vs 매끄러운 함수"를 손바닥으로 그려 보이는 시범. ‘미분’ 단어 안 써도 됨 — '한 점에서 어느 쪽이 내리막인지 잴 수 있다'로 충분.

STEP A3. 경사하강법 — 손실을 줄이는 방향

앱 운영A3 은 ½MSE 규약으로 손실 L = ½·MSE 를 표시. A2 의 MSE 와 같은 모양 곡선이지만 세로축 값만 정확히 절반 — 그 이유는 다음 A4 의 ½ 도입과 같이 보면 한 줄로 풀린다.
A3-1한 점 옆 살짝 — 정답
b = 0.5 → ½ · (0.5 − 1)² = 0.125
b = 1 → ½ · (1 − 1)² = 0
b = 1.5 → ½ · (1.5 − 1)² = 0.125
정답b=0.5 에서 손실을 줄이려면 b를 오른쪽(↑정답 1 쪽). b=1.5 에서는 왼쪽. b=1 에서는 움직이지 않는다 (= 학습 종료 신호).
A3-2갱신 식 빈칸 — 정답
새 w = w − η · ( ∂L/∂w = dw )
새 b = b − η · ( ∂L/∂b = db )
정답기울기 양수(+) 이면 새 w 는 작아진다. 빼기는 곧 "기울기와 반대 방향으로 가라"는 뜻.
A3-34 시나리오 — 정답
조건: 정답 b=1, 시작 b=−2, w=2 고정. 모든 점의 e = b − 1 = −3 → db = 평균(e) = −3. 한 step 새 b = −2 − η·(−3) = −2 + 3η.

A3-4η가 너무 크면 / 너무 작으면 — 정답 예시
정답 예시너무 크면 → 정답을 지나쳐 점점 멀어진다 (발산). 너무 작으면 → 정답에 거의 닿지 못하고 한없이 느림. 딱 좋으면 → 몇 step 만에 손실이 안정적으로 0 근처로. 결국 η 는 한 step 의 보폭.

STEP A4. 기울기 계산하기 — dw·db 유도

A4-1사슬규칙 한 단계씩 — 정답
① L = ½ · (ŷ − y)²   (½ 은 미분 시 2 와 약분되도록)
② ∂L/∂ŷ = (ŷ − y) = e
③ ∂ŷ/∂w = ∂/∂w(w·x + b) = x
④ ∂ŷ/∂b = ∂/∂b(w·x + b) = 1
⑤ ∂L/∂w = e · x,  ∂L/∂b = e · 1 = e
dw = (1/N) · Σ ei · xi
db = (1/N) · Σ ei
수업 팁"사슬규칙" 단어를 두 번 쓰면 충분 — 겉(제곱) 미분 × 속(선형) 미분 = e·x. 더 깊이 안 들어감.

A5-4왜 빼기인가 — 정답 예시
서술 예시기울기는 손실이 커지는 방향을 알려 준다. 줄이려면 반대로 가야 하므로 "− 기울기" 방향으로 한 step 의 보폭(η) 만큼 옮긴다 → 식에 빼기가 들어간다.
A5-5학습 종료 신호 — 정답
새 w = w − η · 0 = w,  새 b = b − η · 0 = b
정답기울기 0 이면 더 이상 움직이지 않는다 → 학습 종료 신호. 이때의 (w, b) 가 손실 곡선의 최저점.

수준특징
상예측·오차·MSE·기울기·학습률·갱신 6 단어를 모두 사용. "예측을 만들고, 정답과의 차이를 숫자(MSE)로 만든 뒤, 손실을 줄이는 방향(기울기)으로 한 step 의 보폭(η) 만큼 w·b 를 옮긴다. 이걸 반복" 의 흐름이 살아 있음.
중5 개 이상 단어 사용. 흐름 중 한두 단계의 인과 연결이 약함 (예: 왜 빼기인가? 가 빠지거나 학습률을 단순 "속도"로만 설명).
하단어 나열에 그치고 흐름이 안 이어짐. 또는 식만 적고 무슨 뜻인지 자기 말로 풀지 못함.
