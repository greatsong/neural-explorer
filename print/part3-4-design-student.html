STEP 1. 도트 그림 학습 — 가중치 시각화의 의미

이 절의 목표
입력이 4개(점수)에서 64개 픽셀(8×8)로 늘어났을 때, 단일 뉴런이 두 그림을 어떻게 구분하는지 — 학습된 가중치 64개를 8×8 그림으로 펼쳐보면 어떤 모양이 나오는지 직접 손으로 발견한다.
픽셀이 입력이 되면   한 픽셀이 켜져 있으면 1, 꺼져 있으면 0. 가중치도 64개. 점수 = Σ 픽셀 × 가중치. 점수 > 0이면 라벨 A, < 0이면 라벨 B.
1-1두 라벨 평균 패턴 — 미리 제공
아래는 두 라벨의 평균 도트 패턴입니다. 라벨 A = "위쪽 가로줄", 라벨 B = "왼쪽 세로줄"의 단순화된 예시.
라벨 A 평균 (위쪽 가로줄)
라벨 B 평균 (왼쪽 세로줄)
1-2가중치 시각화 — A 평균에서 B 평균을 빼면
학습된 가중치를 8×8로 펼치면 "A에서 자주 켜지고 B에서 안 켜지는 자리 = 양수(밝음)", "B에서 자주 켜지고 A에서 안 켜지는 자리 = 음수(어두움)", "둘 다 켜지거나 둘 다 안 켜지는 자리 = 0(회색)"이 됩니다. 1-1의 두 평균을 머릿속으로 빼서 아래 표에 색을 칠하세요.
규칙: A에만 켜진 자리 = +(흰색 비워두기), B에만 켜진 자리 = −(검정 채우기), 둘 다(겹치는 자리) = 0(점 하나만), 둘 다 안 켜짐 = 0(빈칸).
가중치 시각화 (W = A 평균 − B 평균)
1-3새 그림 분류 — 점수 계산
새 그림이 들어왔다고 하자. 점수 = Σ (픽셀 · 가중치). 양수면 A, 음수면 B로 예측. 두 가지 새 그림에 대해 의미를 적어보세요.
새 그림특징점수 부호 (+ / −)예측 (A / B)

왼쪽 두 줄만 켜진 그림B의 음수 영역에 픽셀이 모임

STEP 2. 신경망 설계 — 단계 A·B·C 비교 실험

이 절의 목표
같은 신경망 구조에 분류 클래스 수를 2 → 3 → 10으로 늘릴 때 정확도가 어떻게 달라지는지, 그리고 은닉층 크기를 키우면 어떤 효과가 있는지 결과 표로 비교한다.
은닉층(hidden layer) 등장 — 지금까지는 입력 → 출력 단일 뉴런. 이제 사이에 은닉층을 둔다. 은닉층은 입력의 부분 패턴(예: "왼쪽 위 곡선")을 먼저 잡아내고, 출력층이 그 부분 패턴을 조합해 최종 분류.
▲ 입력 → 은닉층 → 출력층의 다층 신경망 구조
2-1단계 A·B·C 결과 — 비교 표 (은닉 8로 통일)
비교 조건을 똑같이 두기 위해 세 단계 모두 은닉층 1개·은닉 뉴런 8개·15에폭으로 학습한 결과예요(입력 784, 활성화 ReLU, 출력 softmax).
단계분류 클래스파라미터 수시험 정확도
A 🟢0, 1 (2종)6,29899%
B 🟡0, 1, 7 (3종)6,30796%
C 🔴0~9 (10종)6,37082%
표를 보고 빈칸을 채우세요.
클래스 수가 늘어날수록 정확도는 (올라간다 / 내려간다). ☞
파라미터 수는 거의 비슷한데 정확도 차이가 큰 이유: 클래스 수가 많을수록 모델이 (쉬워진다 / 어려워진다). ☞
파라미터 계산 식: (입력 + 1) × 은닉 + (은닉 + 1) × 출력 클래스. 단계 C 검산: (784+1)·8 + (8+1)·10 = + =
앱의 추천 은닉 크기는 단계마다 다릅니다 — A는 4, B는 8, C는 16(난이도가 올라갈수록 추천값이 큼). 활동지 표는 비교를 깔끔하게 하려고 셋 다 8로 맞췄어요. 앱에서 추천값으로 다시 학습시키면 정확도가 더 오릅니다(특히 단계 C).

하지만 4 → 8 → 16 → 32로 두 배씩 키워도 정확도 증가 폭은 (같다 / 점점 작아진다). ☞
관찰 — 정확도 증가가 점점 작아지는 이유를 한 줄로:
2-3왜 1과 7은 헷갈릴까

STEP 3. MNIST 도전 — 더 깊게, 더 다양하게

이 절의 목표
실제 손글씨 데이터(MNIST, 28×28, 10종)에서 모델 구조·학습 설정·데이터 증강이 정확도에 어떻게 영향을 주는지 결과 표로 비교한다.
데이터 증강 (data augmentation)   같은 그림을 회전·이동·축소·전단으로 변형해 데이터를 늘리는 기술. 적은 데이터로도 일반화 능력을 높일 수 있다.
3-1구조와 학습 설정 변화 — 결과 표 (앱의 5단계 프리셋과 매칭)
앱 페이즈 12의 5가지 프리셋 버튼 중 위에서 4개를 학습시킨 결과예요(에폭은 슬라이더로 1~40 사이에서 자유롭게 조정 가능).
프리셋은닉층에폭학습 정확도시험 정확도
🐣 시작[4]370%65%
🌱 작게[8]882%78%
🌳 보통[32]1592%88%
🚀 크게[64, 32]2597%92%
표를 보고 빈칸을 채우세요.
🐣 → 🌱 은닉을 4 → 8로 키우면 시험 정확도 (오른다 / 떨어진다). ☞
🌱 → 🌳 은닉을 8 → 32로, 에폭을 8 → 15로 늘리면 (오른다 / 같다). ☞
🌳 → 🚀 층을 1층 [32]에서 2층 [64, 32]로 깊게 쌓으면 (오른다 / 떨어진다). ☞
🚀에서 학습 정확도(97%)와 시험 정확도(92%)의 격차가 커지는 현상의 이름: (힌트: 2부에서 본 개념)
앱에는 한 단계 더 큰 👑 최강 (128·64 2층·30에폭) 프리셋도 있어요 — 도전해 보고 시험 정확도가 더 오르는지, 아니면 학습/시험 격차만 더 벌어지는지 직접 확인해 보세요.

3-3한계 — 모델이 끝까지 헷갈리는 페어
잘 학습된 모델도 끝까지 헷갈리는 숫자 페어 예: 4-9, 3-8, 5-6. 이 중 하나를 골라, 픽셀 패턴 관점에서 두 숫자가 왜 비슷한지·어떤 점이 다른지 적어보세요.
