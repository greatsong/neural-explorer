이번 STEP의 정체
앱은 "다음 단계 →"를 6번 누르면 자동으로 한 step을 끝냅니다. 이번에는 같은 한 step을 사람의 손으로 처음부터 끝까지 따라가요. 6단계 — 예측 → 오차 → 출력층 기울기 → 은닉층 신호 → 은닉층 기울기 → 갱신 — 의 숫자가 어떻게 한 줄로 이어지는지 직접 확인하세요.
이번 STEP의 모델·데이터·학습률
  • 구조: 입력 1 · 은닉 1 · 출력 1 (가장 단순한 2층 망)
  • 활성화: 은닉층 ReLU, 출력층은 그대로(선형) → ŷ = w₂ · ReLU(w₁·x + b₁) + b₂

  • 초기 가중치: w₁ = 0.5, b₁ = 0.1, w₂ = 1.5, b₂ = 0.0
  • 학습률 η = 0.05
