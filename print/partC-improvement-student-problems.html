STEP C1. 평가와 일반화 (Phase C1)

평가 데이터란?
학습할 때는 안 보여주고, 마지막에 "처음 보는 그림"으로 모델을 시험하는 데이터예요. 학습 데이터의 정확도는 모델이 외운 만큼 높지만, 평가 데이터의 정확도가 진짜 실력입니다.

STEP C2. 외운 정도 = 학습−평가 갭 (Phase C2)

"외운 정도"의 정의
같은 모델·같은 epoch·같은 학습률로 두 번 학습. 단, 학습 데이터 양만 다르게 — 적은 데이터(라벨당 2장 = 4장) vs 전체. 적게 본 모델은 그 몇 장을 통째로 외워버려 학습 정확도는 높지만 평가 정확도가 따라오지 않습니다.
C2-1두 학습 결과 비교 — 직접 옮겨 적기
앱"▶ 두 학습 비교 시작"을 눌러 학습이 끝나면, 좌측 두 카드의 train acc / eval acc를 표에 옮겨 적고 차이 = 외운 정도를 직접 계산하세요.

C2-5갭을 줄이려면 — 세 가지 처방
"외운 정도(갭)를 줄이려면 어떻게 해야 할까?" 세 가지 처방을 떠올려 빈칸을 채우세요. 힌트: 데이터·모델·학습 시간.

  2. 모델을 () — 가중치 수가 적으면 외울 능력 자체가 줄어든다.
  3. 학습을 () — 갭이 벌어지기 전에 멈춘다(조기 종료).
